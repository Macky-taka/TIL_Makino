# node.jp とnpmについて

### node.jp

page作成に使われるjavascriptをサーバー側で動作させるプラットフォーム。本来webブラウザ上で動作するのみでサーバー上では動作しない。Node.jpはjavascript実行環境である。
本来は大量の同時接続をさばけるネットワークアプリケーションの構築が目的のようだ
